FastShop — интернет-магазин обуви, созданный с использованием Next.js и NestJS, построен по архитектуре REST API.

🔑 Аутентификация и безопасность:
Поддержка входа через сторонние сервисы: GitHub и Yandex

Возможность регистрации собственного аккаунта

Двухфакторная аутентификация через Resend (код подтверждения)

Безопасная авторизация с помощью JWT-токенов

🛒 Функциональность магазина
Просмотр и фильтрация товаров по множеству параметров

Добавление товаров в избранное и корзину

Покупка товаров через Stripe

📱 Адаптивный интерфейс
Интерфейс адаптирован под различные устройства — от мобильных телефонов до десктопов.

⚙️ Технологии
Frontend: Next.js, Redux, RTK Query, TailwindCSS, ShadcnUI, Zod, React Hook Form

Backend: NestJS, PostgreSQL, Prisma

Auth: JWT, OAuth (GitHub, Yandex), 2FA через Resend

Payments: Stripe API
